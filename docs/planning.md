User Stories

> As a user I'd like to have an easy to use contact form

> As a user, I want a clean and attractive design so that the website is easy and enjoyable to use.

> As a user, I want the website to be well organized so that I can easily find info.

> As a user, I want to know more about the team so that I understand who created the project.

> As a user, I want to access team member's linkedin profiles so that I can learn a bit more about their experience.

> As a user, I want to see each developer's skills so that I know their strengths and their experience.



MVP

Our MVP is going to be a simple team landing page that displays all the team members with their avatars and clicks to access each individual profile page.
